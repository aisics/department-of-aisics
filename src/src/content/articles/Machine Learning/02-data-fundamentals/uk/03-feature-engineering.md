## –ú–∞—Å—à—Ç–∞–±—É–≤–∞–Ω–Ω—è —Ç–∞ –Ω–æ—Ä–º–∞–ª—ñ–∑–∞—Ü—ñ—è —Ñ—ñ—á - –í–∏—Ä—ñ–≤–Ω—é–≤–∞–Ω–Ω—è —ñ–≥—Ä–æ–≤–æ–≥–æ –ø–æ–ª—è ‚öñÔ∏è

### –ù–∞–≤—ñ—â–æ –º–∞—Å—à—Ç–∞–±—É–≤–∞—Ç–∏ —Ñ—ñ—á—ñ?

–£—è–≤—ñ—Ç—å, —â–æ –≤–∏ –ø–æ—Ä—ñ–≤–Ω—é—î—Ç–µ –¥–≤–æ—Ö —Å—Ç—É–¥–µ–Ω—Ç—ñ–≤:
- –°—Ç—É–¥–µ–Ω—Ç A: –û—Ü—ñ–Ω–∫–∞ –∑ –º–∞—Ç–µ–º–∞—Ç–∏–∫–∏ = 95/100, –ó—Ä—ñ—Å—Ç = 170 —Å–º
- –°—Ç—É–¥–µ–Ω—Ç B: –û—Ü—ñ–Ω–∫–∞ –∑ –º–∞—Ç–µ–º–∞—Ç–∏–∫–∏ = 85/100, –ó—Ä—ñ—Å—Ç = 180 —Å–º

–Ø–∫—â–æ –≤–∏ –∑–∞–ø–∏—Ç–∞—î—Ç–µ "—Ö—Ç–æ –∫—Ä–∞—â–∏–π –∑–∞–≥–∞–ª–æ–º?", –≤–∞—à–∞ –º–æ–¥–µ–ª—å –º–æ–∂–µ –ø–æ–¥—É–º–∞—Ç–∏:
```
A: 95 + 170 = 265 –±–∞–ª—ñ–≤ –∑–∞–≥–∞–ª–æ–º

```

–ó–∞—á–µ–∫–∞–π—Ç–µ, —Ü–µ –Ω—ñ—Å–µ–Ω—ñ—Ç–Ω–∏—Ü—è! –í–∏ —â–æ–π–Ω–æ –¥–æ–¥–∞–ª–∏ —Ç–µ—Å—Ç–æ–≤—ñ –±–∞–ª–∏ –¥–æ –∑—Ä–æ—Å—Ç—É! ü§¶

**–ü—Ä–æ–±–ª–µ–º–∞:** –§—ñ—á—ñ –Ω–∞ —Ä—ñ–∑–Ω–∏—Ö —à–∫–∞–ª–∞—Ö –ø–ª—É—Ç–∞—é—Ç—å –º–æ–¥–µ–ª—ñ.

```
–ù–∞—à—ñ –¥–∞–Ω—ñ –ø—ñ—Ü–µ—Ä—ñ—ó:
- Temperature: -5¬∞C –¥–æ 35¬∞C (–¥—ñ–∞–ø–∞–∑–æ–Ω: 40)
- Price: $12.99 –¥–æ $98.99 (–¥—ñ–∞–ø–∞–∑–æ–Ω: 86)

–ë–µ–∑ –º–∞—Å—à—Ç–∞–±—É–≤–∞–Ω–Ω—è:
–ú–æ–¥–µ–ª—å –¥—É–º–∞—î "price —É 12√ó –≤–∞–∂–ª–∏–≤—ñ—à–∞ –∑–∞ num_pizzas" (–±–æ –±—ñ–ª—å—à—ñ —á–∏—Å–ª–∞!)
```

**–ú–æ–¥–µ–ª—ñ, –Ω–∞ —è–∫—ñ –≤–ø–ª–∏–≤–∞—î –º–∞—Å—à—Ç–∞–±:**
- Linear Regression, Logistic Regression
- Neural Networks
- Support Vector Machines (SVM)
- K-Nearest Neighbors (KNN)
- K-Means Clustering

**–ú–æ–¥–µ–ª—ñ, –Ω–∞ —è–∫—ñ –ù–ï –≤–ø–ª–∏–≤–∞—î:**
- Decision Trees, Random Forest, XGBoost (–≤–æ–Ω–∏ —Ä–æ–∑–±–∏–≤–∞—é—Ç—å –ø–æ –∑–Ω–∞—á–µ–Ω–Ω—è—Ö, –∞ –Ω–µ –≤–µ–ª–∏—á–∏–Ω–∞—Ö)

### 1. Min-Max –º–∞—Å—à—Ç–∞–±—É–≤–∞–Ω–Ω—è (–ù–æ—Ä–º–∞–ª—ñ–∑–∞—Ü—ñ—è)

**–§–æ—Ä–º—É–ª–∞:** `X_scaled = (X - X_min) / (X_max - X_min)`

**–†–µ–∑—É–ª—å—Ç–∞—Ç:** –í—Å—ñ –∑–Ω–∞—á–µ–Ω–Ω—è –º—ñ–∂ 0 —ñ 1

```python
from sklearn.preprocessing import MinMaxScaler

# –ü—Ä–∏–∫–ª–∞–¥: –¢–µ–º–ø–µ—Ä–∞—Ç—É—Ä–∞ –≤—ñ–¥ -5¬∞C –¥–æ 35¬∞C
temps = np.array([[-5], [5], [15], [25], [35]])
```

#### –ö–æ–ª–∏ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ Min-Max –º–∞—Å—à—Ç–∞–±—É–≤–∞–Ω–Ω—è

**‚úÖ –î–æ–±—Ä–µ –¥–ª—è:**
- **Neural Networks** (–±—ñ–ª—å—à—ñ—Å—Ç—å –∞–∫—Ç–∏–≤–∞—Ü—ñ–π –ø—Ä–∞—Ü—é—é—Ç—å –∫—Ä–∞—â–µ –∑ [0, 1] –∞–±–æ [-1, 1])
- **–ó–æ–±—Ä–∞–∂–µ–Ω—å** (–∑–Ω–∞—á–µ–Ω–Ω—è –ø—ñ–∫—Å–µ–ª—ñ–≤ 0-255 ‚Üí 0-1)
- **–ö–æ–ª–∏ –≤–∏ –∑–Ω–∞—î—Ç–µ –º–µ–∂—ñ min/max** (—Ç–µ–º–ø–µ—Ä–∞—Ç—É—Ä–∞, –≤—ñ–¥—Å–æ—Ç–∫–∏)
- **–ê–ª–≥–æ—Ä–∏—Ç–º—ñ–≤ –Ω–∞ –æ—Å–Ω–æ–≤—ñ –≤—ñ–¥—Å—Ç–∞–Ω–µ–π** (KNN, K-Means)

**‚ùå –ü–æ–≥–∞–Ω–æ –¥–ª—è:**
- **–î–∞–Ω–∏—Ö –∑ –≤–∏–∫–∏–¥–∞–º–∏**
  ```python
  # –î–∞–Ω—ñ: [1, 2, 3, 4, 100]
  # –ú–∞—Å—à—Ç–∞–±–æ–≤–∞–Ω–æ: [0, 0.01, 0.02, 0.03, 1.0]
  # –í—Å–µ —Å—Ç–∏—Å–Ω—É—Ç–æ –±—ñ–ª—è 0 —á–µ—Ä–µ–∑ –≤–∏–∫–∏–¥!
  ```
- **–ù–µ–æ–±–º–µ–∂–µ–Ω–∏—Ö —Ñ—ñ—á** (–º–æ–∂–µ –≤–∏–π—Ç–∏ –∑–∞ –º–µ–∂—ñ —Ç—Ä–µ–Ω–æ–≤–∞–Ω–æ–≥–æ –¥—ñ–∞–ø–∞–∑–æ–Ω—É)

### 2. –°—Ç–∞–Ω–¥–∞—Ä—Ç–∏–∑–∞—Ü—ñ—è (Z-Score –Ω–æ—Ä–º–∞–ª—ñ–∑–∞—Ü—ñ—è)

**–§–æ—Ä–º—É–ª–∞:** `X_scaled = (X - mean) / std`

**–†–µ–∑—É–ª—å—Ç–∞—Ç:** –°–µ—Ä–µ–¥–Ω—î = 0, –°—Ç–∞–Ω–¥–∞—Ä—Ç–Ω–µ –≤—ñ–¥—Ö–∏–ª–µ–Ω–Ω—è = 1

```python
from sklearn.preprocessing import StandardScaler

```

#### –ö–æ–ª–∏ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ —Å—Ç–∞–Ω–¥–∞—Ä—Ç–∏–∑–∞—Ü—ñ—é

**‚úÖ –î–æ–±—Ä–µ –¥–ª—è:**
- **–õ—ñ–Ω—ñ–π–Ω–∏—Ö –º–æ–¥–µ–ª–µ–π** (Linear/Logistic Regression, SVM)
- **–î–∞–Ω–∏—Ö –∑ –≤–∏–∫–∏–¥–∞–º–∏** (–º–µ–Ω—à —á—É—Ç–ª–∏–≤–æ –Ω—ñ–∂ min-max)
- **–§—ñ—á, —â–æ —Å–ª—ñ–¥—É—é—Ç—å –Ω–æ—Ä–º–∞–ª—å–Ω–æ–º—É —Ä–æ–∑–ø–æ–¥—ñ–ª—É**
- **–ö–æ–ª–∏ –≤–∏ –Ω–µ –∑–Ω–∞—î—Ç–µ –º–∞–π–±—É—Ç–Ω—ñ–π min/max**

**‚úÖ –ù–∞–π–ø–æ—à–∏—Ä–µ–Ω—ñ—à–∏–π –≤–∏–±—ñ—Ä** –¥–ª—è –∑–∞–≥–∞–ª—å–Ω–æ–≥–æ ML!

### 3. –†–æ–±–∞—Å—Ç–Ω–µ –º–∞—Å—à—Ç–∞–±—É–≤–∞–Ω–Ω—è

**–§–æ—Ä–º—É–ª–∞:** `X_scaled = (X - median) / IQR`

–î–µ IQR = 75-–π –ø–µ—Ä—Å–µ–Ω—Ç–∏–ª—å - 25-–π –ø–µ—Ä—Å–µ–Ω—Ç–∏–ª—å

**–†–µ–∑—É–ª—å—Ç–∞—Ç:** –í–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—î –º–µ–¥—ñ–∞–Ω—É —Ç–∞ IQR –∑–∞–º—ñ—Å—Ç—å —Å–µ—Ä–µ–¥–Ω—å–æ–≥–æ —Ç–∞ std (—Å—Ç—ñ–π–∫–µ –¥–æ –≤–∏–∫–∏–¥—ñ–≤!)


#### –ö–æ–ª–∏ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ —Ä–æ–±–∞—Å—Ç–Ω–µ –º–∞—Å—à—Ç–∞–±—É–≤–∞–Ω–Ω—è

- **–°–∫–æ—à–µ–Ω–∏—Ö —Ä–æ–∑–ø–æ–¥—ñ–ª—ñ–≤**
- **–ö–æ–ª–∏ –≤–∏ —Ö–æ—á–µ—Ç–µ —Å—Ç–∞–±—ñ–ª—å–Ω–æ—Å—Ç—ñ**

### –ü—Ä–∏–∫–ª–∞–¥ –ø–æ—Ä—ñ–≤–Ω—è–Ω–Ω—è –º–∞—Å—à—Ç–∞–±—É–≤–∞–Ω–Ω—è

```python
# –î–∞–Ω—ñ –∑ –≤–∏–∫–∏–¥–æ–º: [1, 2, 3, 4, 100]
```

### –ö—Ä–∏—Ç–∏—á–Ω–µ –ø–æ–ø–µ—Ä–µ–¥–∂–µ–Ω–Ω—è: –£–Ω–∏–∫–∞–π—Ç–µ –≤–∏—Ç–æ–∫—É –¥–∞–Ω–∏—Ö!

**‚ùå –ù–ï–ü–†–ê–í–ò–õ–¨–ù–û:** –§—ñ—Ç—É—î–º–æ –Ω–∞ –≤–µ—Å—å –¥–∞—Ç–∞—Å–µ—Ç, –ø–æ—Ç—ñ–º —Ä–æ–∑–¥—ñ–ª—è—î–º–æ

```python

```
